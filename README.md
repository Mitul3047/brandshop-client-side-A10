# Brand-Based Website with MERN Stack

This project aims to develop a brand-based website using the MERN stack (MongoDB, Express, React, Node.js).

## Project Overview

### Category Chosen: Technology and Electronics

The website showcases products from renowned brands within the Technology and Electronics category, including but not limited to Apple, Samsung, Sony, Google, and Intel.

## Main Features

- **Navigation:** Navbar with website name/logo, Home, Add Product, My Cart, and Login.
- **Home Page:** Navbar, banner, footer, brand sections (6 types), and 2 extra sections.
- **Add Product Page:** Form for users to add a product with specific details.
- **Brand Page:** Redirects to a page displaying brand-related products and advertisements.
- **Product Detail Route:** Detailed product information with an Add to Cart button.
- **Cart Page:** Displays added products with the option to delete.
- **Update Product Route:** Allows updating product information through a form.
- **Authentication:** Email/password-based login and registration, additional social login options.
- **Error Handling:** Display errors for registration and login based on specified criteria.
- **404 Page:** Not found page for invalid routes.

## Bonus Features

- **Git Commits & README:** Minimum 10 meaningful commits (client-side) and 5 commits (server-side), detailed project description in the README with the client-side live link.
- **Persistent Private Routes:** Retains user login after page reload.
- **Responsive Design:** Ensures compatibility across desktop and mobile devices.
- **Theme Toggle:** Optional dark/light theme toggle for the home page.
- **Hosting:** Deploy server-side on Vercel and client-side on Firebase.
